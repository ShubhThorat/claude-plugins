# apple-home

Control Apple HomeKit devices from Claude Code.

## Tools

- `home_devices` — list all devices and rooms
- `home_control` — turn a device on/off (`home_control Floor Lamp on`)
- `home_scenes` — list HomeKit scenes
- `home_run_scene` — run a scene
- `home_setup_guide` — setup instructions for Shortcuts

## Setup

```
/plugin install apple-home@shubhthorat
```

Control uses macOS Shortcuts — run `home_setup_guide` after install.
