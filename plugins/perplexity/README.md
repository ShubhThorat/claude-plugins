# Perplexity Claude plugin

Production-ready Perplexity MCP plugin for Claude using:

- `npx -y @perplexity-ai/mcp-server`

## Documentation Index

Fetch the complete documentation index at:

- [https://docs.perplexity.ai/llms.txt](https://docs.perplexity.ai/llms.txt)

Use this file to discover all available pages before exploring further.

## Prerequisites

- `npx` available locally
- `PERPLEXITY_API_KEY` set in your environment

## Setup

```bash
export PERPLEXITY_API_KEY="your_key_here"
```

Then install/reload plugins in Claude.

## Available tools

- `perplexity_search`
- `perplexity_ask`
- `perplexity_research`
- `perplexity_reason`
